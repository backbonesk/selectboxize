selectboxize

A simple JavaScript library for styling selectboxes.

Selectboxize

Selectboxize is a simple and lightweight JavaScript library for styling and replacing default selectboxes. The library is heavily based on previous work by Krzysztof Suszyński.

Regular selectbox

$('#countries-1').selectboxize();

Restricted width

The width is restricted by applying width: 150px on the <select> definition in HTML.

$('#countries-2').selectboxize();

Restricted number of items rolled-out

When selectbox is clicked and rolled out only 5 items are visible (default is 15).

$('#countries-3').selectboxize({
  'listboxMaxSize': 5
});

Callback upon a value selection

The following example will display an alert modal dialog upon selecting a value.

$('#countries-4').selectboxize({
  'callback': function(item, e) {
    alert("You have selected country '" + $(item).text() + "' with value of '" + $(item).data('value') + "'.");
  }
});
